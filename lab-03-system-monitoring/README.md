Linux System Monitoring Lab
Overview

System monitoring is a critical responsibility of Linux system administrators and DevOps engineers. Monitoring system resources helps detect performance issues, prevent outages, and maintain server health.

In this lab, I built a Linux system health monitoring script that collects important system metrics and generates a structured health report.

The script automates the process of checking:

System uptime

Disk usage

Memory utilization

High CPU processes

Active users on the system

This demonstrates practical Linux administration and automation skills


Technologies Used

Linux CLI

Bash Scripting

System Monitoring Commands

Key Linux commands used:

uptime

df -h

free -h

ps

who

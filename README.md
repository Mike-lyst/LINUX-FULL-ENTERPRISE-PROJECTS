# LINUX-FULL-ENTERPRISE-PROJECTS
This repository documents a hands-on Linux enterprise lab environment built to simulate real-world system administration tasks performed by IT and DevOps teams.

The goal of this lab is to move beyond theoretical learning and develop practical operational experience with Linux systems used in enterprise environments such as fintech, telecom, and cloud infrastructure.

The lab simulates a small organization with multiple departments and focuses on how system administrators manage users, access control, automation, security, monitoring, and web infrastructure.

Through a series of structured labs, I practice common responsibilities performed by Linux administrators, including:

User and group management for multiple departments

Role-based access control and permission management

SSH hardening and secure authentication

Bash scripting for system automation

Scheduled task automation using cron

System monitoring for CPU, memory, disk, and uptime

Log management and troubleshooting

Firewall configuration and security hardening

Fail2Ban automation to protect services

Nginx web server configuration and reverse proxy setup

The lab environment simulates 25 users across different departments, allowing realistic testing of access control, automation scripts, and operational workflows.

Each lab folder includes:

A detailed explanation of the task

Commands executed during the lab

Screenshots of terminal output

Documentation of key concepts and lessons learned

This project is part of my journey toward becoming a Linux/Cloud Infrastructure Engineer, with a focus on building real operational skills that apply to production environments.

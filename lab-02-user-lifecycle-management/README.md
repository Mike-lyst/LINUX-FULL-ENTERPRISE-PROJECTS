This lab simulates the user lifecycle management process in an enterprise Linux environment.

In real organizations, IT teams must manage users from the moment they join the company until they leave. This process is called the User Lifecycle and includes:

Provisioning – creating new user accounts and assigning access

Auditing – reviewing existing users and permissions

Deprovisioning – removing user access when employees leave

This lab demonstrates how Bash scripting can automate these processes, making system administration more efficient and secure.
